ISOQOL PROMIS global screening demo

This demo illustrates how the PROMIS Global Health can be used to screen for follow-up domains to assess using CAT measures.
The demo consists of three sections:
  • Screeneing criteria questions: The filtering configuration of the global health takes place here. (Upper left-hand section)
  • Patient's scheduled quesstionnaired: This is the real time assessment list for the patient. (Upper right-hand section)
  • Assessment section: The patient view of the demo. (Bottom half of the screen)
No data is stored in this demo. This is only for illustrating the assessment workflow. To continue with the demo ackowledge that no data is available.
